# 会議アクション抽出 Skill

会議メモからアクションアイテムを抽出し、担当者・期限付きでリスト化します。

## 抽出ルール

以下のパターンをアクションとして認識：
- 「〜する」「〜してください」「〜をお願い」
- 「TODO」「タスク」「アクション」
- 担当者名 + 動詞（「田中さんが確認」など）

## 出力フォーマット

```markdown
## アクションアイテム

| # | アクション | 担当 | 期限 |
|---|-----------|------|------|
| 1 | 〇〇を確認する | 田中 | 今週中 |
| 2 | △△を作成する | 鈴木 | 12/5 |
```

## 使用例

入力:
```
今日の会議で、田中さんがAPIの仕様を確認することになった。
鈴木さんは金曜までにデザイン案を作成。
来週のミーティングで進捗確認する。
```

出力:
```markdown
## アクションアイテム

| # | アクション | 担当 | 期限 |
|---|-----------|------|------|
| 1 | APIの仕様を確認する | 田中 | - |
| 2 | デザイン案を作成する | 鈴木 | 金曜 |
| 3 | 進捗確認ミーティング | 全員 | 来週 |
```
